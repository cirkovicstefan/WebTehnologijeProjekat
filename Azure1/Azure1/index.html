Microsoft Azure

Microsoft Azure (ranije poznat kao Windows Azure) servis je za računarstvo u oblaku koji je stvorio Majkrosoft za pravljenje, testiranje, raspoređivanje i upravljanje aplikacijama i servisima kroz centre podataka kojima upravlja Majkrosoft.

Microsoft Azure Servisi

Azure koristi virtuelizaciju velikih razmera u Majkrosoftovim centrima podataka širom sveta i nudi više od 600 usluga.
Računarski servisi

Virtuelne mašine, infrastruktura kao usluga (IaaS) koja omogućava korisnicima pokretanje virtuelnih mašina Majkrosoft Vindovs i Linuks opšte namene, kao i unapred konfigurisane slike mašina za popularne softverske pakete.

Većina korisnika koristi Linuks na Azureu, neke od mnogih ponuđenih Linuks distribucija, uključujući Majkrosoftovu sopstvenu Azure Sphere zasnovanu na Linuksu. Usluge aplikacija, okruženje platforme kao usluge (PaaS) omogućava programerima lako objavljivanje i upravljanje veb lokacijama.

Veb lokacije, Azure veb lokacije omogućavaju programerima da grade veb lokacije koristeći ASP.NET, PHP, Node.js ili Pajton i mogu se primeniti pomoću FTP, Git, Merkurial, Tim Foundacijski Server ili otpremiti putem korisničkog portala. Ova funkcija je najavljena u obliku pregleda u junu 2012. godine na sastanku Meet Majkrosoft Azur.

Kupci mogu da kreiraju veb lokacije u PHP-u, ASP.NET-u, Node.js-u ili Pajtonu ili da izaberu nekoliko aplikacija otvorenog koda iz galerije za primenu. Ovo uključuje jedan aspekt ponude platforme kao usluge (PaaS) za platformu Majkrosoft Azur. Preimenovan je u Veb Aps u aprilu 2015. Veb Poslovi, aplikacije koje se mogu primeniti u okruženje usluge aplikacija za primenu pozadinske obrade koja se može pozivati po rasporedu, na zahtev ili se neprekidno pokretati. Usluge The Blob, Table i Queue mogu se koristiti za komunikaciju između Veb Aplikacija i Veb Poslova i za pružanje stanja.
Memorijski servisi

Usluge skladištenja pružaju REST i SDK API-je za čuvanje i pristup podacima u oblaku. Usluga tabela omogućava programima da skladište strukturirani tekst u particioniranim kolekcijama entiteta kojima se pristupa particijskim ključem i primarnim ključem.

Azure Tabela usluga je NoSql nerelaciona baza podataka. Blob usluga omogućava programima da skladište nestrukturirani tekst i binarne podatke kao blobove kojima se može pristupiti HTTP (S) putanjom. Blob usluga takođe pruža sigurnosne mehanizme za kontrolu pristupa podacima. Usluga čekanja omogućava programima da asinhrono komuniciraju porukom koristeći redove.
Mobilni servisi

Mobile Engagement collects prikuplja analitiku u stvarnom vremenu koja ističe ponašanje korisnika. Takođe pruža push obaveštenja za mobilne uređaje.

HockeyApp se može koristiti za razvoj, distribuciju i beta testiranje mobilnih aplikacija.
Upravljanje podacima

Azure Data Explorer pruža analitiku velikih podataka i mogućnosti istraživanja podataka. Azure Search pruža pretragu teksta i podskup strukturiranih filtera OData’s pomoću API-ja REST ili SDK.

Azure SQL baza podataka radi na stvaranju, skaliranju i proširivanju aplikacija u oblak pomoću tehnologije Majkrosoft SQL Server. Takođe se integriše sa Active Directory i Microsoft System Center i Hadoop.
Razmena poruka

Azure Service Bus omogućava aplikacijama pokrenutim u Azureovim prostorijama ili na drugim uređajima da komuniciraju sa Azureom. Ovo pomaže u izgradnji skalabilnih i pouzdanih aplikacija u uslužno orijentisanoj arhitekturi (SOA).

Azure servisna magistrala podržava četiri različite vrste komunikacionih mehanizama:

1. Event Hubs,
2. Queues,
3. Topics,
4. Relays
CDN

Globalna mreža za isporuku sadržaja (CDN) za audio, video, aplikacije, slike i druge statičke datoteke. Može se koristiti za keširanje statičke imovine veb lokacija geografski bliže korisnicima radi povećanja performansi.

Mrežom može da upravlja HTTP API zasnovan na REST-u. Azur ima 94 lokacije prisutnosti širom sveta (poznate i kao Edge lokacije) od aprila 2020.
