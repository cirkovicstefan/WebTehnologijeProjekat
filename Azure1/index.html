Microsoft Azure

Microsoft Azure (ranije poznat kao Windows Azure) servis je za računarstvo u oblaku koji je stvorio Majkrosoft za pravljenje, testiranje, raspoređivanje i upravljanje aplikacijama i servisima kroz centre podataka kojima upravlja Majkrosoft.

Microsoft Azure Servisi

Azure koristi virtuelizaciju velikih razmera u Majkrosoftovim centrima podataka širom sveta i nudi više od 600 usluga.

Računarski servisi

Virtuelne mašine, infrastruktura kao usluga (IaaS) koja omogućava korisnicima pokretanje virtuelnih mašina Majkrosoft Vindovs i Linuks opšte namene, kao i unapred konfigurisane slike mašina za popularne softverske pakete.

Mobilni servisi

Mobile Engagement collects prikuplja analitiku u stvarnom vremenu koja ističe ponašanje korisnika. Takođe pruža push obaveštenja za mobilne uređaje. HockeyApp se može koristiti za razvoj, distribuciju i beta testiranje mobilnih aplikacija.

Memorijski servisi

Usluge skladištenja pružaju REST i SDK API-je za čuvanje i pristup podacima u oblaku. Usluga tabela omogućava programima da skladište strukturirani tekst u particioniranim kolekcijama entiteta kojima se pristupa particijskim i primarnim ključem.

Upravljanje podacima

Azure Data Explorer pruža analitiku velikih podataka i istraživanja mogućnosti istraživanja podataka. Azure Search pruža pretragu teksta i podskup filtera OData’s pomoću API-ja REST ili SDK.

Razmena poruka

Azure Service Bus omogućava aplikacijama pokrenutim u Azureovim prostorijama ili na drugim uređajima da komuniciraju sa Azureom. Ovo pomaže u izgradnji skalabilnih i pouzdanih aplikacija u uslužno orijentisanoj arhitekturi (SOA).

CDN

Globalna mreža za isporuku sadržaja (CDN) za audio, video, aplikacije, slike i druge statičke datoteke. Može se koristiti za keširanje statičke imovine veb lokacija geografski bliže korisnicima radi povećanja performansi.

Microsoft Azure dizajn

Microsoft Azure koristi specijalizovani operativni sistem, nazvan Microsoft Azure, za pokretanje svog „sloja“. Klaster hostovan u Majkrosoftovim centrima podataka koji upravljaju računarstvom i resursima za skladištenje računara i obezbeđuje resurse (ili njihov podskup) aplikacijama pokrenutim na vrhu Microsoft Azure-a. Microsoft Azure je opisan kao „oblačni sloj“ povrh brojnih sistema Vindovs Server, koji koriste Vindovs Server 2008 i prilagođenu verziju Hiper-V, poznatu kao Microsoft Azure Hypervisor za pružanje virtuelizacije usluga.

Skaliranjem i pouzdanošću kontroliše Microsoft Azure Fabric Controller-a, koji osigurava da usluge i okruženje ne propadnu ako jedan ili više servera otkaže u Majkrosoftovom centru podataka, a koji takođe omogućava upravljanje korisnikovom veb aplikacijom, kao što je dodeljivanje memorije i balansiranje opterećenja.

Azur pruža API izgrađen na REST-u, HTTP-u i XML-u koji programeru omogućava interakciju sa uslugama koje pruža Microsoft Azure. Majkrosoft takođe nudi klijentsku biblioteku klasa kojom se objedinjuju funkcije interakcije sa uslugama. Takođe se integriše sa Microsoft Visual Studio, Git, and Eclipse.

Pored interakcije sa uslugama putem API-ja, korisnici mogu da upravljaju Azur uslugama i pomoću veb portala Azur, koji je opštu dostupnost dostigao u decembru 2015. Portal omogućava korisnicima da pretražuju aktivne resurse, menjaju postavke, pokreću nove resurse i pregledaju osnovne podatke o nadgledanju sa aktivnih virtuelnih mašina i usluga.
